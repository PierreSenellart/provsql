\mainpage ProvSQL C/C++ API

This is the internal C/C++ API of the ProvSQL PostgreSQL extension.
The PostgreSQL-facing interface is written in C, while circuit
manipulation, graph algorithms, and semiring evaluation use C++.

See the **[Classes](annotated.html)** page for circuit representations
and semirings, **[Files](files.html)** for per-file documentation,
or **[Namespaces](namespaces.html)** for the `semiring` namespace.
